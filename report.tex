\section{Introduction}
This short report introduces two fundamental probability distributions:
the standard normal distribution and a binomial distribution with parameters
$n=10$ and $p=0.3$.  
Both graphs were generated using Python.

\section{Standard Normal Distribution}
The standard normal distribution is defined as:
\[
f(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}.
\]

\section{Binomial Distribution}
The binomial distribution with parameters $n=10$ and $p=0.3$ has probability mass function:
\[
P(X=k) = \binom{10}{k} (0.3)^k (0.7)^{10-k}, \quad k=0,\dots,10.
\]
